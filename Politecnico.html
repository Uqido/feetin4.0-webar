The printer is an electronic output device of the computer, capable of printing the data chosen by the user on paper. The computer is therefore able to transfer images and / or text onto paper through the printer.
The monitor is an electronic device for displaying electronically transmitted images, text and video. It is considered an output device, but if it has the touch screen function it also becomes an input device, as it can perform the function of keyboard and mouse.
The mouse is a portable pointing device (or peripheral) that detects two-dimensional movement with respect to a flat surface; this movement is translated into the movement of a pointer on a monitor, which allows a control of the graphical interface of a computer. The mouse has one or more buttons for various operations, and to which various input functions can be assigned.
The keyboard is a computer input device intended for manually entering data, numeric or alphanumeric, into the computer memory and controlling the computer itself.
A central processing unit (CPU), is the electronic circuitry that executes instructions comprising a computer program. The CPU performs basic arithmetic, logic, controlling, and input/output (I/O) operations specified by the instructions in the program. This contrasts with external components such as main memory and I/O circuitry, and specialized processors such as graphics processing units (GPUs).
Random-access memory (RAM) is a form of computer memory that can be read and changed in any order, typically used to store working data. A random-access memory device allows data items to be read or written in almost the same amount of time irrespective of the physical location of data inside the memory.
